# 👨🏾‍💻 Flávio Almeida

**`Software Engineer | Java • Spring Boot • Cloud • React • DevOps`**

Me chamo Flávio Almeida, sou Desenvolvedor de Software e estudante de Sistemas de Informação no Instituto Federal de Sergipe (IFS).

Atualmente atuo como **Team Leader e Desenvolvedor Full Stack na InfoEduc**, liderando iniciativas de desenvolvimento de software e participando da construção de soluções escaláveis para o ecossistema educacional da empresa.

Tenho foco principal em **Java e Spring Boot**, trabalhando com APIs REST, microsserviços, arquitetura de software, autenticação e integração entre sistemas. Também possuo experiência com **React, TypeScript, Docker, AWS, Cloudflare, PostgreSQL, MongoDB, Kubernetes, Bubble.io e n8n**.

Recentemente participei da arquitetura e desenvolvimento do **InfoEduc One Account**, plataforma centralizada de identidade baseada em **OIDC, Zitadel, Supabase, Fastify, Cloudflare e infraestrutura cloud**, permitindo autenticação única para múltiplos produtos da empresa.

Sou apaixonado por tecnologia, arquitetura de software, cloud computing e liderança técnica, buscando constantemente evoluir minhas habilidades para construir sistemas modernos, resilientes e de alta performance.

# 🏗️ Projeto de Destaque

## 🔐 InfoEduc One Account

Plataforma centralizada de identidade para o ecossistema InfoEduc, inspirada nos modelos utilizados por Microsoft, Google e Atlassian.

### Tecnologias Utilizadas

- OIDC
- OAuth2
- Zitadel
- React
- TypeScript
- Fastify
- PostgreSQL
- Supabase
- Docker
- Cloudflare Pages
- Cloudflare DNS
- GitHub Actions
- VPS Linux
- Nginx
- Brevo
- Doppler

### Principais Funcionalidades

- Single Sign-On (SSO)
- Login centralizado
- Cadastro unificado
- Recuperação de senha
- Integração entre múltiplos produtos
- Gestão de usuários
- Segurança baseada em OIDC
- Deploy automatizado
