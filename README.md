# 👨🏾‍💻 Flávio Almeida

**`Software Engineer | Java • Spring Boot • React • Cloud • DevOps`**

Me chamo Flávio Almeida e sou Desenvolvedor de Software e estudante de Sistemas de Informação no Instituto Federal de Sergipe (IFS). Atualmente atuo como Team Leader e Desenvolvedor Full Stack na InfoEduc, participando da construção de soluções para o setor educacional com foco em Java, Spring Boot, React e arquitetura de software.

Tenho experiência no desenvolvimento de APIs REST, microsserviços, autenticação centralizada (OIDC/OAuth2), cloud computing, Docker, AWS, Cloudflare, PostgreSQL, MongoDB e Kubernetes. Também atuo com automações utilizando n8n e desenvolvimento No-Code com Bubble.io.

Recentemente participei da arquitetura e desenvolvimento do InfoEduc One Account, plataforma de identidade centralizada baseada em padrões modernos de autenticação e infraestrutura cloud.
